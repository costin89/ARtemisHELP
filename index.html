ARtemis Anleitung

- Benutzeroberfläche | Touchgesten | Gamecontroller | Import -

Diese Anleitung beschreibt die grundlegenden Funktionen der Benutzeroberfläche von ARtemis. Die Benutzeroberfläche ermöglicht es den Nutzern, verschiedene Objekte in eine AR-Szene zu laden und diese zu steuern. Diese Anleitung bietet einen Überblick über die grundlegenden Funktionen der Benutzeroberfläche, der Touchgesten, der Gamecontroller funktionen sowie den Import von USDZ-Dateien.

1. Navigation und Steuerung

1.1 Plus-Button

Der Plus-Button dient zur Aktivierung der Navigationsleiste. Bei Betätigung dieses Buttons wird die Navigationsleiste sichtbar und ermöglicht den Zugriff auf weitere Funktionen der Anwendung.

1.2 X-Button

Der X-Button hat die gegenteilige Funktion des Plus-Buttons. Durch das Drücken des X-Buttons wird die zuvor geöffnete Navigationsleiste wieder ausgeblendet. Dies sorgt für eine aufgeräumte Benutzeroberfläche und ermöglicht eine ungestörte Nutzung der AR-Szene.

1.3 Up-Button und Down-Button

Die Up- und Down-Buttons sind essenziell für das Laden von Objekten in die AR-Szene:

  • Up-Button: Mit dem Up-Button können Nutzer Objekte in die AR-Szene laden.
  • Down-Button: Der Down-Button ermöglicht das Laden von Objekten in die AR-Szene auf einer niedrigeren Ebene.

Diese Buttons ermöglichen den Import und Verwaltung der Objekte innerhalb der AR-Szene.

1.4 I-Button

Der I-Button dient zum Öffnen des Handbuchs der Anwendung. Durch das Drücken dieses Buttons wird ein Benutzerhandbuch angezeigt, welches detaillierte Anweisungen und Informationen zur Nutzung der verschiedenen Funktionen sowie Funktionen der Benutzeroberfläche enthält.

2. Touch-Gestenerkennung in ARtemis auf dem iPad / iPhone

2.1 Positionierung

Um ein AR-Objekt auf einem iPad zu bewegen, wird in der Regel eine einfache Tap Geste verwendet. Ein Tap auf die Bodenfläche (mit dem Symbol [ + ] ) aktiviert es für Bewegungen. Vorwärts und rückwärts Bewegung: Leichtes Ziehen mit einem Finger in die entsprechende Richtung auf dem Bildschirm. Seitliche Bewegungen: Leichtes Ziehen mit einem Finger seitwärts, um das Objekt nach rechts oder links zu verschieben.

2.2 Rotation

Für die Rotation eines AR-Objekts um seine eigene Achse wird häufig die Zwei-Finger-Rotationsgeste genutzt: platzieren Sie zwei Finger auf dem Display und drehen Sie diese entweder im Uhrzeigersinn oder gegen den Uhrzeigersinn, um das Objekt entsprechend zu rotieren.

2.3 Skalierung

Die Größenänderung eines AR-Objekts erfolgt über die Pinch-Geste, die mit zwei Fingern ausgeführt wird: führen Sie die Finger zusammen, um das Objekt zu verkleinern, oder spreizen Sie die Finger, um es zu vergrößern.

2.4 Reset (Zurücksetzen)

Ein Doppel-Tap auf das AR-Objekt kann verwendet werden, um dessen Ausrichtung und Skalierung auf den Ursprungszustand zurückzusetzen: ein schnelles doppeltes Tippen auf das iPad Display setzt dieses in seinen Anfangszustand zurück.

3. Gamecontroller Funktionen in ARtemis auf dem iPad / iPhone

3.1 Left Thumbstick (linker Analogstick)

Der linke Analogstick wird verwendet, um die Positionierung in der AR-Szene zu steuern. Durch Vorwärts- und Rückwärtsbewegungen des Sticks wird die Bewegung vorwärts bzw. rückwärts ermöglicht, während seitliche Bewegungen die Objekte nach links oder rechts verschieben.

Beispielbild

3.2 A- und B-Tasten [X- und Kreis-Taste]

Die A- und B-Tasten [X- und Kreis-Taste] werden zur Skalierung der Objekte verwendet. Durch Drücken der A-Taste [X-Taste] wird die Skalierung des ausgewählten Objekts vergrößert, während die B-Taste [Kreis-Taste] die Skalierung verkleinert. Dies ermöglicht dem Benutzer eine einfache und intuitive Anpassung der Größe der Objekte.

Beispielbild

3.3 X-Taste [Quadrat-Taste]

Die X-Taste [Quadrat-Taste] dient als Reset-Funktion, um die Position, Rotation und Skalierung der Objekte auf ihre Ausgangswerte zurückzusetzen. Dies ermöglicht dem Benutzer eine schnelle Neuausrichtung oder Anpassung der Objekte in der Szene.

Beispielbild

3.4 Left-Right Trigger (linke-rechte Abzugstasten)

Die linken und rechten Abzugstasten dienen zur Rotation der Objekte um ihre eigene Achse. Durch Drücken des linken Triggers wird das ausgewählte Objekt gegen den Uhrzeigersinn gedreht, während der rechte Trigger eine Drehung im Uhrzeigersinn bewirkt.

Beispielbild

3.5 Left-Right Shoulder Button (linke-rechte Schultertasten)

Die linken und rechten Schultertasten dienen zur Auswahl und zum Laden von Objekten in die AR-Szene. Durch Drücken der linken Schultertaste wird ein Objekt ausgewählt, während die rechte Schultertaste das Laden des ausgewählten Objekts in die Szene auslöst.

Beispielbild

3.6 Menü-Taste (Icon mit drei horizontalen Linien "Hamburger-Menü")

Die Menü-Taste des Game-Controllers kann die Darstellung der Hilfe-Datei in der Applikation aufrufen. Dieser Vorgang ermöglicht den Benutzer:innen den direkten Zugriff auf unterstützende Dokumentationen und Anleitungen, ohne die Anwendung verlassen zu müssen.

4. Bereitstellung einer USDZ-Datei auf einem iPad / iPhone

Durch die Nutzung der Files-App, des iTunes Shared Folders oder durch direktes Herunterladen von einem Server können Sie USDZ-Dateien effizient auf Ihr iPad übertragen und für die ARtemis-App zugänglich machen. Die folgenden 3 Methoden bieten flexible Lösungen für die Dateiverwaltung und ermöglichen eine nahtlose Integration von .usdz-Dateien. Im Folgenden wird auf drei unterschiedlichen Wegen beschrieben, wie man eine USDZ-Datei auf ein iPad überträgt und zugänglich macht: 1) durch die Files-App, 2) den iTunes Shared Folder oder 3) den direkten Download von einem Server. Bei der Nutzung des iTunes Shared Folders werden die Daten auch direkt in die Files-App übertragen.

Methode 1: Übertragung über die Files-App

  1. USDZ-Datei vorbereiten

    Stellen Sie sicher, dass die USDZ-Datei auf Ihrem Computer gespeichert und leicht zugänglich ist.

  2. Dateiübertragung mittels AirDrop
    1. Aktivieren Sie AirDrop auf Ihrem iPad und Ihrem Mac.
    2. Suchen Sie auf Ihrem Mac die USDZ-Datei und klicken Sie mit der rechten Maustaste darauf.
    3. Wählen Sie im Kontextmenü „Teilen“ und dann „AirDrop“.
    4. Wählen Sie Ihr iPad aus der Liste der verfügbaren Geräte.
    5. Bestätigen Sie den Empfang der Datei auf Ihrem iPad.
  3. Datei im Files-App anzeigen
    1. Öffnen Sie die Files-App auf Ihrem iPad.
    2. Navigieren Sie zum Ordner „Auf meinem iPad“ oder zu einem anderen spezifischen Speicherort, an dem die Datei gespeichert wurde. Meistens finden Sie die USDZ-Datei im "Download" Ordner.
    3. Die USDZ-Datei sollte nun sichtbar und zugänglich sein.
    4. Tippen Sie auf „Auswählen“ oben rechts.
    5. Tippen Sie auf die Datei, die Sie verschieben möchten. Ein Häkchen erscheint neben der ausgewählten Datei.
    6. Tippen Sie auf das Bewegen-Symbol am unteren Bildschirmrand.
    7. Navigieren Sie zu dem ARtemis_TUDa-Ordner, in den Sie die Datei verschieben möchten.
    8. Tippen Sie auf „Bewegen“ oben rechts, um die Datei in den ARtemis_TUDa-Ordner zu verschieben.

    Alternativ: Drag and Drop

    1. Öffnen Sie die Files-App und gehen Sie zum Ursprungsordner.
    2. Halten Sie die Datei gedrückt, bis sie sich abhebt.
    3. Ziehen Sie die Datei mit Ihrem Finger in den gewünschten Zielordner „ARtemis_TUDa“.
    4. Lassen Sie die Datei los.

    Tipps:

    • Sie können auch mehrere Dateien gleichzeitig verschieben, indem mehrere Dateien ausgewählt werden, bevor Sie auf „Verschieben“ tippen.
    • Um schneller zu navigieren, können Sie die Suchleiste oben verwenden, um den „ARtemis_TUDa“ Zielordner zu finden.

Methode 2: Übertragung über den iTunes Shared Folder

  1. iTunes herunterladen und installieren

    Stellen Sie sicher, dass iTunes auf Ihrem Computer installiert ist. Laden Sie es von der offiziellen Apple-Website herunter und installieren Sie es, falls noch nicht geschehen.

  2. iPad verbinden

    Verbinden Sie Ihr iPad über ein USB-Kabel mit Ihrem Computer. Öffnen Sie iTunes und wählen Sie Ihr iPad aus, sobald es erkannt wird. Dieser Arbeitsschritt ist z.Z. für Mac gültig.

  3. Dateifreigabe einrichten

    Klicken Sie in iTunes auf das Symbol Ihres iPads und navigieren Sie zu „Dateifreigabe“.

    Wählen Sie die „ARtemis_TUDa“-App aus der Liste, die diese Dateiübertragung unterstützt.

  4. USDZ-Datei hinzufügen

    Übertragen Sie per Drag and Drop die .usdz-Datei in den ausgewählten App-Ordner auf Ihr iPad.

    Die Datei wird automatisch mit der Files-App synchronisiert.

Methode 3: Direktes Laden von einem Server über die Files-App

  1. Serverzugang einrichten

    Stellen Sie sicher, dass die USDZ-Datei auf einem Server gespeichert ist und Sie über die Zugangsdaten verfügen.

  2. Files-App öffnen

    Öffnen Sie die Files-App auf Ihrem iPad.

  3. Serververbindung herstellen

    1. Tippen Sie auf das Menü-Symbol oben links.
    2. Wählen Sie „Mit Server verbinden“.
    3. Geben Sie die Serveradresse ein und tippen Sie auf „Verbinden“.
    4. Geben Sie Ihre Anmeldedaten ein, falls erforderlich.

  4. USDZ-Datei herunterladen

    1. Navigieren Sie im Server-Verzeichnis zur USDZ-Datei.
    2. Tippen Sie auf die Datei, um sie herunterzuladen.
    3. Die Datei wird nun in der Files-App gespeichert und ist zugänglich.
    4. Tippen Sie auf „Auswählen“ oben rechts.
    5. Tippen Sie auf die Datei, die Sie verschieben möchten. Ein Häkchen erscheint neben der ausgewählten Datei.
    6. Tippen Sie auf das Bewegen-Symbol am unteren Bildschirmrand.
    7. Navigieren Sie zu dem ARtemis_TUDa-Ordner, in den Sie die Datei verschieben möchten.
    8. Tippen Sie auf „Bewegen“ oben rechts, um die Datei in den ARtemis_TUDa-Ordner zu verschieben.
